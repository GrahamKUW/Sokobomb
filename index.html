• SOKOBOMB

    • Created by Kavon Graham

    • Sokobomb is a Sokobon style game where you must push bombs onto defusal squares in a certain amount of moves!

    • What is a Sokobon-style game you might be wondering?

      A Sokoban-style game is a puzzle game where you move a character around a grid to push

      objects (like crates) into specific target spots, using careful planning since objects can't be pulled—only pushed.

    • CONTROLS

      • WASD or ARROW KEYS to move

      • LMB to interact with User Interface
